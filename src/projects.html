Sentiment Analysis

This project scans and gives reviews a positive, negative, or neutral score based on common positive and nagetive words. The project uses OOP design and implementation. Each review is read in from a file, uses a Map to keep track of the words and their occurances, a Set to keep track of all the positve/negative words, and can list the highest and lowest scoring reviews.
